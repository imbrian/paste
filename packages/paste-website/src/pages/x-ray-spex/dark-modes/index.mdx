---
title: Dark and Light modes for Paste via Themes
description: Paste Themes can be pretty versatile. In addition to allowing global themes for customers, they can serve UI needs such as a light and dark mode too. We've created a quick demo to show how Paste might be used to create a dark mode for our UIs. As "dark mode" is just a theme object switch, the dark mode can be activated via a user control, or can respect a time or OS settings. The example shown highlights an encapsulated light and dark mode on the same page, and outlines the few changes needed to achieve this.
---
